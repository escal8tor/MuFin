# MuFin
An unofficial Jellyfin client for MuOS.

## Features
- Watch meida on your MuOS device. 
- And little else (for now).

## Installation
1. Download the latest `muxapp` from releases.
2. Copy it to your device under the `ARCHIVE` directory. 
3. On the device, go to *Apps* > *Archive Manager*, and select *MuFin* to install.
4. Once the installation finishes, back to *Apps*, then *Mufin* to launch.

## Configuration
The configuration is stored under `/mnt/union/MUOS/application/MuFin/config.ini`.
The commented template can be found under `Mufin\app\res\static\config.ini`.
